📊 SNS 성과 리포트 대시보드

터미널 명령어 대신 이 화면에서 버튼으로 실행할 수 있게 만든 도구
⚠️ 이 화면은 이 컴퓨터 안에서만 동작해요. 다른 컴퓨터·팀원은 이 주소로 못 들어오고, 각자 자기 컴퓨터에서 이 프로그램을 따로 실행해야 해요. 로그인 정보도 이 컴퓨터 밖으로 전송되지 않고 여기 파일로만 저장돼요.

1. 로그인 상태 확인

실적을 수집하려면 트위터/인스타그램에 로그인된 상태(세션)가 필요해요. 한 번 로그인해두면 며칠~몇 주 정도는 계속 쓸 수 있고, 풀리면 로그인 화면이 계속 뜨는 식으로 알 수 있어요 — 그럴 때 여기서 다시 로그인하면 돼요.

트위터: 확인 중...
인스타그램: 확인 중...

버튼을 누르면 이 컴퓨터에 실제 브라우저 창이 새로 뜰 거예요. 그 창에서 평소처럼 직접 로그인(2단계 인증 있으면 그것도 그대로)하고, 다 되면 아래 "로그인 완료했어요" 버튼을 눌러주세요.

2. 오늘 실적 수집

자사(PW)·경쟁사(BH)의 오늘 하루 트위터·인스타그램 게시물을 모아서 비교 리포트를 만들어요. 실제로 인터넷을 오가며 게시물을 읽어오는 거라 보통 몇 분 정도 걸려요.

3. 방금 수집한 데이터로 리포트만 다시 만들기

재수집(몇 분) 없이, 방금 수집해둔 원본 데이터로 엑셀·화면 리포트만 몇 초 안에 다시 만들어요. 수동으로 매칭을 조정했거나 리포트 형식만 다시 보고 싶을 때 사용해요.

4. 날짜가 떨어진 여러 기간 비교하기

예: 6/10~13, 6/18~22처럼 사이에 공백이 있는 여러 기간을 각각 "오늘 실적 수집"으로 미리 모아뒀다면, 그 기간들을 나란히 비교하는 표를 만들어요. 아래에 이미 수집해둔 기간을 "시작일_종료일" 형태로 한 줄씩 적어주세요(2개 이상).

5. 특정 계정 하나만 단독으로 보기

자사·경쟁사 비교가 아니라, 임의의 트위터 계정 하나만 골라서 그 계정 게시물 성과만 순위표로 보고 싶을 때 사용해요.

6. 노션으로 보내기

방금 만든 리포트의 표를 스크린샷이 아니라 진짜 노션 표로 만들어줘요(검색·정렬·편집 가능).

복사하기: 버튼 누르면 바로 클립보드에 복사돼요 — 노션 페이지에서 Ctrl+V만 누르면 끝(연동 설정 필요 없음).
자동 전송: 노션 연동을 미리 설정해뒀다면 클릭 한 번으로 지정한 페이지에 바로 생성돼요 — notion-config.json 파일이 없으면 실패 메시지가 떠요.
