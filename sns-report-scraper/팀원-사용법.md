# SNS 리포트 도구 사용법 (팀원용)

처음 쓰는 분도 그대로 따라오면 됩니다. **아무것도 설치할 필요 없어요.**

---

## 처음 한 번

1. **다운로드**
   허브 페이지의 "SNS 성과 리포트 대시보드 — 다운로드" 버튼을 눌러 ZIP 파일을 받습니다.

2. **압축 풀기**
   받은 ZIP을 원하는 위치(예: 바탕화면)에 압축 해제합니다.

3. **실행**
   압축 푼 폴더 안의 **`start-dashboard.bat`** 을 더블클릭합니다.
   - 검은 창이 하나 뜨고, 잠시 뒤 **브라우저가 자동으로 열리면서** 대시보드 화면이 나옵니다.
   - ⚠️ 그 검은 창은 도구가 켜져 있는 동안 계속 열어두세요(창을 닫으면 도구도 꺼집니다).

4. **로그인** (처음 한 번만)
   대시보드 화면에서 트위터/인스타그램 각각 **"로그인하기"** 버튼을 누릅니다.
   - 로그인용 브라우저 창이 뜨면 **직접 로그인**하세요.
   - 로그인을 마치면 대시보드의 **"로그인 완료했어요"** 버튼을 누릅니다.
   - 이 로그인 정보는 **본인 컴퓨터 안에만** 저장되고 어디로도 전송되지 않습니다.

5. **사용**
   "오늘 하루 수집" 등의 버튼으로 실적을 수집하고, 아래 목록에서 리포트를 내려받을 수 있습니다.

---

## 나중에 업데이트할 때

도구가 개선되면 폴더 안의 **`update.bat`** 을 더블클릭하기만 하면 됩니다.
- 바뀐 부분만 몇 초 만에 자동으로 받아옵니다(다시 다운로드할 필요 없음).
- 그다음 평소처럼 `start-dashboard.bat` 을 실행하면 최신 버전으로 열립니다.

> 가끔 "새 버전(v2.4 등)이 나왔다"고 안내를 받으면, 그때만 허브 페이지에서 ZIP을 다시 받아
> 압축을 풀면 됩니다.

---

## 잘 안 될 때

- 브라우저가 자동으로 안 열리면, 주소창에 직접 `http://localhost:4848` 을 입력하세요.
- 검은 창에 빨간 글씨가 뜨거나 멈추면, 그 창을 캡처해서 관리자에게 보내주세요.
