# 이벤트 추첨기

이벤트 참가자 엑셀/CSV를 업로드하고, 이번 회차 조건을 자유텍스트로 입력하면
LLM이 조건에 맞는 후보를 골라주고, 그중 최종 당첨자는 코드가 실제 난수로
추첨하는 도구. 기획 배경은 [`PLAN.md`](./PLAN.md) 참고.

## 지금 상태 (진행 중)

- **1단계(파일 업로드) · 3단계(추첨)는 지금 바로 테스트 가능** — `index.html`을
  브라우저로 열어서 엑셀/CSV 올리고, "필터링 없이 전체 참가자로 진행"으로
  추첨 버튼까지 눌러볼 수 있다.
- **2단계(LLM 필터링)는 아직 작동 안 함** — `api/filter.js`가 Vercel에 배포되고
  `GEMINI_API_KEY` 환경변수가 설정돼야 실제로 동작한다. 그 전에는 "필터링
  실행"을 누르면 안내 메시지가 뜨고, 대신 "전체 참가자로 진행" 버튼으로
  추첨 기능만 미리 확인할 수 있다.

## 사용법

1. `index.html`에서 참가자 엑셀(.xlsx) 또는 CSV 파일 업로드 (첫 줄 = 헤더)
2. 이번 회차 조건을 문장으로 입력 (예: "사진 첨부한 사람 + 나루토를 언급한
   사람 + 정성스럽게 쓴 사람") → **필터링 실행**
3. 후보 명단이 나오면 당첨 인원수를 정하고 **추첨하기** — 코드가 실제 난수로
   뽑는다 (LLM은 후보 선정까지만 관여, 최종 당첨자 선택에는 관여하지 않음)
4. **결과 다운로드**로 당첨자 명단 + 이번 조건만 파일로 저장 (원본 참가자
   데이터·댓글은 포함하지 않음)

## Vercel 배포 시 참고

- 이 저장소는 GitHub Pages로도 쓰이고 있어서, Vercel 프로젝트를 만들 때
  **Root Directory를 `event-raffle`로 지정**해야 한다 (레포 전체가 아니라
  이 폴더만 배포).
- 배포 후 프로젝트 환경변수에 `GEMINI_API_KEY`를 등록해야 `api/filter.js`가
  동작한다.
