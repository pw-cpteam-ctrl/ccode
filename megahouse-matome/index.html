셀 위 드래그=위치 조정 · 스크롤=확대. 12장 초과 시 자동으로 2페이지.

🖼 사진 올리기
① 사진 수정 섹션 열기 → 사진 업로드 영역 클릭하거나 파일을 드래그
② 사진은 자동으로 현재 탭 페이지에 추가됩니다
③ 12장 초과 시 다음 페이지로 자동 분리
✏️ 제품명 입력
제품명 일괄입력 박스에 한 줄에 하나씩 입력 후 적용 버튼
또는 미리보기 포스터의 제품명을 직접 클릭해서 수정 가능
🖱 사진 위치·크기 조정
미리보기 포스터의 사진 위에서
· 드래그 → 위치 이동
· 스크롤(휠) → 확대/축소
↕ 순서 변경 / 삭제
사진 수정 패널 좌측 ::번호 를 드래그해서 순서 변경
우측 ✕ 버튼으로 삭제, 썸네일 호버 시 사진교체 버튼 표시
🔖 재판 / 특전 뱃지
각 제품 행의 재판 체크 → 재판 뱃지 표시
특전 체크 → 초회특전 뱃지 표시
뱃지 색상은 레이아웃 수정 → 재판 배지 에서 변경
💬 메모 추가
각 제품 행의 메모 체크 → 입력칸 표시
글자크기·색상·위치(위/옆) 설정 가능
🏷 오픈용 헤더
레이아웃 수정 섹션 안 오픈용 헤더 항목에서 체크 → 기존 파란 헤더 대신 검정 그라데이션 오버레이로 전환
타이틀·날짜·스토어명 문구/크기/굵기 조정 가능
스토어명 박스 색(흑백)·투명도, 그라데이션 농도·퍼짐 세부 조절 가능
💾 저장 / 다운로드
작업 내용은 자동 저장 (브라우저 새로고침해도 유지)
하단 개별 다운로드 → 현재 페이지 PNG
하단 전체 다운로드 → 모든 페이지 PNG 일괄 저장
