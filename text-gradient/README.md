# 텍스트 그라데이션 이미지 생성기

배지 문구 / 좌우 타이틀(`|` 구분) / 불릿 리스트를 입력하면, 검정 그라데이션이
깔린 **투명 배경 PNG(1080×1350)** 를 만들어 주는 도구입니다. 빌드/서버 없이
`index.html`을 브라우저로 바로 엽니다.

## 사용법

1. 왼쪽 패널에 배지 문구·타이틀(좌/우)·불릿 리스트(줄바꿈 구분)를 입력한다.
2. 그라데이션 시작점/퍼짐/농도 슬라이더로 하단 그라데이션을 조절한다.
3. **PNG 다운로드 (투명)** 클릭 → `slot{n}.png` 저장 (배경 투명, 1080×1350 고정).

## 마크다운 일괄입력

여러 슬롯을 한 번에 초안으로 뽑고 싶을 때 사용한다.

```
- 주술회전 | 메가캣 프로젝트
  - 주술냥코 시부야 사변 (재판)
  - 유지/메구미/노바라
  - 고죠/나나미/스쿠나
- 진격의 거인 | 스케일 피규어
  - 에렌 예거 & 리바이 병장
```

- **최상위 불릿(`-`) = 슬롯 1개.** 텍스트에 `|`가 있으면 좌/우 타이틀로 자동 분리.
- **들여쓴 하위 불릿 = 그 슬롯의 불릿 리스트.**
- **배지 문구는 "공통 배지 문구" 입력값이 생성되는 슬롯 전체에 그대로 적용**된다
  (마크다운 자체엔 배지를 넣을 자리가 없음).
- **일괄 적용을 누르면 기존 슬롯은 전부 지워지고 새로 만들어진다** — 잘못 눌렀으면
  Ctrl+Z로 복구.
- 슬롯 개수 제한은 없다. 다만 **100개 초과 시 "계속할까요?" 확인창**이 한 번 뜬다
  (하드 제한 아님, 실수로 대량 텍스트를 잘못 붙여넣었을 때 대비용).
- 그라데이션(시작점/퍼짐/농도) 설정은 마크다운에 없으므로, 일괄 적용 직전에
  보고 있던 슬롯의 그라데이션 값이 새로 생성되는 슬롯 전체에 그대로 유지된다.
- 일괄 생성은 초안 용도이므로, 2단 배치처럼 슬롯마다 다른 세부 조정은 생성 후
  슬롯별로 직접 손본다.

## 배치 · 크기

배지 / 타이틀 / 불릿을 각각 따로 위(Y)/아래로 옮기고 글자 크기를 조절할 수 있다
(슬라이더 6개: 배지 위치·크기, 타이틀 위치·크기, 불릿 위치·크기). 슬롯마다 독립
적으로 저장되므로, 마크다운으로 일괄 생성한 뒤 특정 슬롯만 배치를 다르게 손봐도
다른 슬롯에는 영향이 없다.

## 저장 슬롯

- 상단 탭에서 슬롯 전환·추가(＋, 개수 제한 없음)·삭제(×, 슬롯이 2개 이상일 때만) 가능.
- 슬롯 탭 클릭 시 해당 슬롯 위치로 부드럽게 스크롤 이동.
- 모든 입력은 **IndexedDB에 자동 저장**되어 새로고침해도 유지된다.
- **Ctrl+Z**로 실행취소 (최대 30단계, 입력창 포커스 중에는 비활성).

## 미리보기 배치 (가로 일렬 ↔ 바둑판)

스테이지 상단 토글로 두 방식을 오갈 수 있다. **기본값은 바둑판**이며, 창 너비에
따라 한 줄에 보이는 열 개수가 자동으로 바뀐다(인스타/윈도우 탐색기 그리드와
동일한 방식 — 넓히면 더 많이, 좁히면 더 적게). 선택한 모드는 IndexedDB에
저장되어 새로고침해도 유지된다.

## 구현 메모

- 화면 미리보기는 `.canvas`(1080×1350 원본) 에 `transform:scale(0.3)`을 걸어
  `.canvas-clip`(축소 후 실제 크기) 안에 담아 보여준다. PNG로 내보낼 때는
  이 transform을 잠깐 제거하고 원본 크기 그대로 캡처한 뒤 복원한다 — 이 처리를
  빠뜨리면 캡처 결과가 캔버스 좌상단 한쪽에 쪼그라들어 찍히니 주의.
