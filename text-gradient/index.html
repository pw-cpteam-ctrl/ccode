인디자인 파일목록 생성기
• 파일명을 한 줄에 하나씩 입력. 확장자 생략 시 .png 자동 추가.
• [그룹제목.png] 줄을 넣으면 그 아래 사진들이 한 그룹으로 묶이고, 그룹의 첫 번째 사진에만 투명제목 이미지가 매칭된다 (인디자인에서 폴더 맨 앞장에만 제목을 붙이는 방식과 동일) — 그룹 사용 시 아래 "투명제목 폴더 경로"도 필수.
• 캐릭터별 하위 폴더에서 자동 인식(Chrome/Edge 전용) = 상위 폴더를 고르면 폴더마다 "표지"가 들어간 사진(없으면 이름순 첫 장)을 그룹 맨 앞에 놓고 파일명 목록을 자동으로 채워준다. 단, 브라우저 보안상 실제 절대경로는 읽을 수 없어서, 다운로드용 "사진 폴더 경로"는 이 기능과 별개로 여전히 직접 입력해야 한다.
• 다운로드 결과는 UTF-16LE(BOM) 텍스트라 인디자인 데이터 병합 가져오기에 바로 쓸 수 있다.
