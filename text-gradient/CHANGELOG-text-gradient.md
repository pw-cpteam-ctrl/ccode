# CHANGELOG — 텍스트 그라데이션 이미지 생성기 (text-gradient)

> 작성 규칙은 루트 [`CHANGELOG-작성-규칙.md`](../CHANGELOG-작성-규칙.md) 참고.

---

## 2026-07-03 (V1.0 초기 버전)
- **핵심 기능**
  - 배지 문구 / 좌우 타이틀 / 불릿 리스트를 입력하면 하단 그라데이션이 깔린 투명 배경 PNG를 만드는 도구 최초 추가
    (기술: 단일 `index.html` 정적 도구, 캔버스 1080×1350 고정, `html-to-image` CDN으로 PNG 캡처)
  - 여러 슬롯을 한 번에 만드는 마크다운 일괄입력 기능 추가
    (기술: 최상위 불릿=슬롯, 들여쓴 하위 불릿=불릿리스트로 파싱, `|` 문자로 좌/우 타이틀 자동 분리)
  - 창 너비에 따라 열 개수가 자동으로 바뀌는 반응형 바둑판 미리보기 추가
    (기술: `#slotsRow.grid-mode` flex-wrap, 가로 일렬/바둑판 토글)

- **배치·크기**
  - 배지/타이틀/불릿 각각 위치(Y)·크기 슬라이더 최초 추가
    (기술: 슬롯별 `badgeY`/`titleY`/`bulletsY` 절대 좌표 필드, `position:absolute` + `top` 직접 지정)
  - 배치·크기 슬라이더를 2열 → 3열 레이아웃으로 정리
    (기술: `.row3` flex 레이아웃으로 배지/타이틀/불릿 컬럼 병렬 배치)

## 2026-07-04 (V1.1 ~ V1.3 인디자인 파일목록 생성기 추가)
- **인디자인 파일목록 생성기 (신규 기능)**
  - 어도비 인디자인 데이터 병합용 파일명 목록을 자동 생성하는 유틸리티 추가
    (기술: 파일명 줄바꿈 입력 → UTF-16LE(BOM) 텍스트로 다운로드, `<@사진1>` 헤더 형식)
  - `[제목파일명]` 줄로 사진을 그룹 지정하고, 그룹 첫 장에만 투명제목 이미지를 매칭하는 기능 추가
    (기술: 그룹 파싱 로직, 그룹당 첫 항목만 2번째 탭 열에 제목 경로 삽입)
  - 캐릭터별 하위 폴더에서 파일 목록을 자동 인식하는 기능 추가 (Chrome/Edge 전용)
    (기술: File System Access API `showDirectoryPicker`, "표지" 포함 파일 자동 감지 후 그룹 첫 장 배치)
  - 그룹 제목 로직 버그 수정
    (기술: 그룹 첫 항목 판별 조건 보정)

## 2026-07-06 (V1.4 ~ V1.7 마크다운 볼드 헤더, 그라데이션 보강, 자간 슬라이더)
- **마크다운 일괄입력 개선**
  - `**시리즈 | 상품유형**` 볼드 헤더 형식 지원 추가
    (기술: 헤더+비들여쓰기 불릿 조합 파싱 시 캐릭터 단위로 쪼개지던 버그 해결, 헤더 유무로 파서 자동 분기)

- **그라데이션**
  - 그라데이션에 "압축" 옵션 추가 (시작점/퍼짐/농도/압축 4종 구성)
    (기술: 오버레이 `top` 오프셋으로 밴드 높이 조절)
  - 그라데이션 슬라이더를 2열×2행 그리드 레이아웃으로 변경
    (기술: `.row2` 2단 배치로 가독성 개선)

- **배치·크기**
  - 배치·크기 슬라이더를 "개별 조정"/"전체 적용" 두 그룹으로 분리
    (기술: `ALL_LAYOUT_FIELDS` 매핑으로 전체 슬롯 일괄 반영 로직 추가)
  - 타이틀·불릿 자간(letter-spacing) 슬라이더 추가, 기본값 -50으로 타이트하게 변경
    (기술: 슬라이더 값(-100~100)을 1000으로 나눈 em 단위로 `letter-spacing` 적용)
  - 배지 배경 기본 오퍼시티 90%, 타이틀·불릿 텍스트 그림자 추가
    (기술: `rgba(0,0,0,.9)` 배경, `text-shadow` 이중 레이어로 밝은 사진 위 가독성 개선)
  - 인디자인 파일목록 생성기에 사용법 도움말 박스 추가
    (기술: 그룹 지정 방식·자동 인식 한계·다운로드 인코딩 설명을 패널에 인라인 표기)

## 2026-07-08 (V1.8 불릿 좌/우 2열 입력 및 마커 기반 표시)
- **불릿 리스트**
  - 불릿을 좌열/우열 2열로 나눠 입력하고 이미지에도 2열로 반영하는 기능 추가
    (기술: `bulletsR` 필드 추가, `.bullets` CSS Grid 2단 컬럼)
  - 줄 앞에 "-" 마커가 있는 줄만 `•` 불릿으로 표시, 없으면 일반 텍스트로 표시하도록 변경
    (기술: 모든 줄에 강제로 `• ` 접두어를 붙이던 방식을 마커 감지 정규식으로 교체, 마크다운 일괄입력 결과도 "-" 재부착으로 하위호환 유지)

- **미리보기**
  - 미리보기 슬롯 캔버스를 클릭하면 그 슬롯이 편집 패널로 바로 전환되는 기능 추가
    (기술: 슬롯 카드 `click` 이벤트로 `curSlot` 전환, 활성 슬롯 파란 테두리 표시)

## 2026-07-09 (V1.9 ~ V2.1 불릿 2열 UX 개선, 하단 기준 자동 정렬, 버그 수정)
- **불릿 2열 UX 개선**
  - 불릿 2열을 기본 OFF로 변경, "2열 사용" 체크박스로 선택 시에만 활성화
    (기술: 2열이 항상 켜져 있어 우열이 비어도 좌열 폭이 절반으로 줄던 문제 해결, `bulletsTwoCol` 플래그 추가)
  - 좌:우 폭 비율 조절 슬라이더 추가 (기본 50:50)
    (기술: `bulletsColRatio` 필드, `grid-template-columns: Nfr Mfr` 동적 지정)
  - 좌열/우열 입력창 사이에 드래그 가능한 손잡이 추가해 폭 비율을 직접 조절 가능하게 함
    (기술: 입력창 사이 `.col-divider` 드래그로 `bulletsColRatio` 실시간 갱신, 패널 슬라이더와 양방향 동기화)
  - 미리보기 이미지 위에 있던 2열 폭 조절 드래그바 제거
    (기술: 이미지를 가려 거슬린다는 피드백에 따라 캔버스 오버레이 리사이저 삭제, 좌측 패널 손잡이로 통합)

- **UI 정리**
  - 도구 내 모든 슬라이더/드래그바를 얇은 형태로 통일
    (기술: `input[type=range]` 트랙 4px + 원형 손잡이 13px로 커스텀 스타일링, 좌/우열 구분 손잡이도 10px→5px로 축소)
  - "개별 조정"/"전체 적용" 안내 문구 가시성 개선
    (기술: 흐린 회색 캡션을 파란 배지 스타일(`.subsec`)로 변경)

- **개별 다운로드 (신규 기능)**
  - 슬롯 카드마다 "⬇ 다운로드" 버튼을 추가해 슬롯 전환 없이 바로 그 슬롯만 PNG로 저장 가능하게 함
    (기술: `downloadPng(pg)`에 슬롯 번호 파라미터 추가, 기존 패널 버튼은 인자 없이 현재 슬롯 유지)

- **서브타이틀 (신규 기능)**
  - 타이틀과 불릿 리스트 사이에 한 줄짜리 서브타이틀 입력칸 추가
    (기술: `subtitle` 필드, 개별 조정/전체 적용/자간 슬라이더까지 타이틀·불릿과 동일 패턴 지원)

- **하단 기준 자동 정렬 (구조 변경)**
  - 배지/타이틀/서브타이틀/불릿의 고정 절대 Y좌표 방식을 폐기하고, "하단 기준선(Y)" 한 지점을 바닥으로 불릿→서브타이틀→타이틀→배지 순으로 쌓아 올리는 자동 정렬로 전환
    (기술: `layoutBottomAnchoredStack()` — 각 요소의 렌더링된 실제 높이를 `getBoundingClientRect()`로 측정해 역순으로 위치 계산, 내용 없는 요소는 스킵)
  - 기존 "위치(Y)" 슬라이더를 절대좌표에서 "자동 위치 대비 오프셋"으로 재활용 (기본값 0)
    (기술: 옛 절대좌표 필드(`badgeY` 등)는 그대로 두고 신규 오프셋 필드(`badgeYOffset` 등)를 별도로 둬서 기존 슬롯도 안전하게 오프셋 0부터 시작하도록 하위호환 처리)

- **버그 수정**
  - "전체 적용"의 위치 오프셋 4개 슬라이더가 전혀 반영되지 않던 문제 수정
    (기술: 개별 슬라이더 동기화 시 상태 필드명(`badgeYOffset`)을 DOM id로 오인해 조회 실패 → 예외 발생으로 렌더링이 중단되던 버그, all-id에서 개별 id를 유도하는 방식으로 수정)
