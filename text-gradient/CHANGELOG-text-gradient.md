## 2026-07-23 (V2.2 ~ V2.6 사진 일괄 합성 도구(batch-merge) 신규 추가)

### 사진 일괄 합성 도구 (신규 기능)
- 엑셀/CSV 매칭표 + 사진 + 투명 이미지를 올리면 1080×1350 캔버스에 자동 합성하는 도구 추가
  (기술: `text-gradient/batch-merge.html` 신규 파일, Canvas 2D `drawImage` 기반 합성 — 라이브 텍스트가 없어 html-to-image 미사용)
- 사진 비율이 제각각이어도 꽉 채워 중앙 자르기(cover-fit) 기본 적용, 필요한 사진만 확대/축소·드래그로 수동 조정 가능
  (기술: cover-fit 배율 계산 후 `scaleMul`/`ox`/`oy`로 오프셋 적용, 축소로 여백 생기면 배경색으로 채움)
- 배경색을 흰색 또는 사진에서 뽑은 주요색으로 자동 지정하는 기능 추가
  (기술: 24×24 축소 캔버스 픽셀 평균값으로 대표색 추출)
- 결과 전체를 zip 파일 하나로 일괄 다운로드하는 기능 추가
  (기술: JSZip, 파일명 중복 시 순번 접미사 자동 부여, 진행률 표시)

### 저장 안정성
- 새로고침해도 작업 중이던 조정 내용이 그대로 복원되도록 개선
  (기술: IndexedDB에 원본 파일은 매칭 직후 1회만 저장하고 줌/위치/배경색 조정값은 400ms 디바운스로 저장 분리, localStorage는 조정값 백업 폴백으로만 사용)
- 아직 사진에 배정되지 않은 투명 이미지 후보까지 새로고침 후 유지되도록 수정
  (기술: 오버레이 파일 풀을 별도 캐시 키(`overlay_pool`)로 분리 저장 — 기존엔 이미 배정된 것만 복원되던 구멍 수정)

### 입력 편의성 (신규 기능)
- 엑셀 첫 줄이 헤더여도 자동으로 인식해서 건너뛰는 기능 추가
  (기술: 첫 줄이 업로드 파일과 매칭 안 되고 확장자 형태도 아니면 헤더로 판단해 자동 스킵)
- 사진 폴더를 통째로 선택하면 엑셀 없이 바로 아트보드+리스트가 만들어지는 기능 추가
  (기술: `webkitdirectory` input, 자연정렬(natural sort)로 리스트 순서 고정 — 이후 사진 추가돼도 기존 줄 안 흔들림)
- 배정 리스트에서 투명 이미지명을 엑셀처럼 드래그로 여러 줄에 한 번에 채우는 기능 추가
  (기술: 채우기 손잡이(fill-handle) 드래그로 범위 계산 후 값 일괄 복사, "적용" 버튼을 눌러야만 아트보드에 반영)
- 미리보기 크기를 슬라이더로 조절하는 기능 추가
  (기술: 썸네일 캔버스를 물리 해상도까지 다시 그려서 확대해도 안 뭉개지게 처리)
- 미리보기에서 마우스 휠로 개별 아트보드를 바로 확대/축소하는 옵션 추가
  (기술: 체크박스로 그리드 `wheel` 이벤트의 `preventDefault` 여부 전환, 클릭 시 상세 편집창은 항상 열림)

### UI 정리
- 사진 업로드 칸이 두 개(엑셀용/폴더용)라 헷갈리던 문제 수정
  (기술: 투명 이미지 업로드를 공통 섹션으로 분리, 나머지를 "방법 A(엑셀)"/"방법 B(폴더, 추천)"로 명확히 구분)

## 2026-07-09 (V1.9 ~ V2.1 불릿 2열 UX 개선, 하단 기준 자동 정렬, 버그 수정)
- **불릿 2열 UX 개선**
  - 불릿 2열을 기본 OFF로 변경, "2열 사용" 체크박스로 선택 시에만 활성화
    (기술: 2열이 항상 켜져 있어 우열이 비어도 좌열 폭이 절반으로 줄던 문제 해결, `bulletsTwoCol` 플래그 추가)
  - 좌:우 폭 비율 조절 슬라이더 추가 (기본 50:50)
    (기술: `bulletsColRatio` 필드, `grid-template-columns: Nfr Mfr` 동적 지정)
  - 좌열/우열 입력창 사이에 드래그 가능한 손잡이 추가해 폭 비율을 직접 조절 가능하게 함
    (기술: 입력창 사이 `.col-divider` 드래그로 `bulletsColRatio` 실시간 갱신, 패널 슬라이더와 양방향 동기화)
  - 미리보기 이미지 위에 있던 2열 폭 조절 드래그바 제거
    (기술: 이미지를 가려 거슬린다는 피드백에 따라 캔버스 오버레이 리사이저 삭제, 좌측 패널 손잡이로 통합)

- **UI 정리**
  - 도구 내 모든 슬라이더/드래그바를 얇은 형태로 통일
    (기술: `input[type=range]` 트랙 4px + 원형 손잡이 13px로 커스텀 스타일링, 좌/우열 구분 손잡이도 10px→5px로 축소)
  - "개별 조정"/"전체 적용" 안내 문구 가시성 개선
    (기술: 흐린 회색 캡션을 파란 배지 스타일(`.subsec`)로 변경)

- **개별 다운로드 (신규 기능)**
  - 슬롯 카드마다 "⬇ 다운로드" 버튼을 추가해 슬롯 전환 없이 바로 그 슬롯만 PNG로 저장 가능하게 함
    (기술: `downloadPng(pg)`에 슬롯 번호 파라미터 추가, 기존 패널 버튼은 인자 없이 현재 슬롯 유지)

- **서브타이틀 (신규 기능)**
  - 타이틀과 불릿 리스트 사이에 한 줄짜리 서브타이틀 입력칸 추가
    (기술: `subtitle` 필드, 개별 조정/전체 적용/자간 슬라이더까지 타이틀·불릿과 동일 패턴 지원)

- **하단 기준 자동 정렬 (구조 변경)**
  - 배지/타이틀/서브타이틀/불릿의 고정 절대 Y좌표 방식을 폐기하고, "하단 기준선(Y)" 한 지점을 바닥으로 불릿→서브타이틀→타이틀→배지 순으로 쌓아 올리는 자동 정렬로 전환
    (기술: `layoutBottomAnchoredStack()` — 각 요소의 렌더링된 실제 높이를 `getBoundingClientRect()`로 측정해 역순으로 위치 계산, 내용 없는 요소는 스킵)
  - 기존 "위치(Y)" 슬라이더를 절대좌표에서 "자동 위치 대비 오프셋"으로 재활용 (기본값 0)
    (기술: 옛 절대좌표 필드(`badgeY` 등)는 그대로 두고 신규 오프셋 필드(`badgeYOffset` 등)를 별도로 둬서 기존 슬롯도 안전하게 오프셋 0부터 시작하도록 하위호환 처리)

- **버그 수정**
  - "전체 적용"의 위치 오프셋 4개 슬라이더가 전혀 반영되지 않던 문제 수정
    (기술: 개별 슬라이더 동기화 시 상태 필드명(`badgeYOffset`)을 DOM id로 오인해 조회 실패 → 예외 발생으로 렌더링이 중단되던 버그, all-id에서 개별 id를 유도하는 방식으로 수정)
