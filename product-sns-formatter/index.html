상품 SNS 포맷터

오늘 작업할 상품을 목록에 담고, 상품마다 ① 정보 입력 → ② 원고 생성 → ③ 캐러셀 이미지 합성까지 한 화면에서 끝내는 도구예요.

📌 처음 쓰시는 분은 여기부터 읽어주세요

  1. ① 왼쪽 "+ 상품 추가"로 오늘 작업할 상품을 목록에 담으세요. 상품마다 상단에서 브랜드를 고를 수 있어요.
  2. ② 오른쪽 상세에서 상품정보/캐릭터 메모를 입력하고 "원고 생성"을 누르면 몇 초 안에 SNS 원고(X용, 실제 링크 포함)가 나와요.
  3. ③ "인스타용 복사"는 그 원고에서 링크 줄만 안내문구로 자동 치환해서 복사해줘요. 미리보기 없이 바로 클립보드에 담겨요.
  4. ④ 아래로 내려가면 캐러셀 이미지 칸이 있어요. 사진을 올리면 자동으로 4:5 비율로 채워 잘리고, 실제 주황 틀을 얹은 완성본을 만들 수 있어요.
💡 알아두면 좋은 것 1 — 이 도구는 "기억력"이 없어요. 원고 생성 버튼을 누를 때마다 완전히 새로 시작해요. "이전에 말한 거 기억해서" 같은 건 안 통해요.
💡 알아두면 좋은 것 2 — 작업 내용(상품 목록, 입력, 원고, 이미지)은 이 브라우저에 자동 저장돼요. 페이지를 다시 열어도 그대로 남아있어요. (다른 기기/브라우저로는 안 옮겨져요)
💡 알아두면 좋은 것 3 — 결과물 포맷이 마음에 안 들면 "개선사항 메모장"에 적어주세요! AI한테 전달되는 게 아니라, 사람이 나중에 확인하고 규칙을 고칠지 판단하는 용도예요.
왼쪽에서 "+ 상품 추가"를 눌러 시작하세요.

✓ 오늘 작업 전체 내려받기

담아둔 상품들의 완성 이미지를 한 번에 ZIP으로, 원고는 한 번에 복사해서 가져갈 수 있어요.

✎개선사항 메모장

여기 적는 내용은 LLM에 절대 전송되지 않습니다. 사람이 나중에 훑어보고 규칙 파일에 반영할지 판단하는 용도예요.

🪲 이상 감지 로그 (AI가 규칙 어겨서 자동 차단된 기록)

일본어 무단 생성처럼 기계적으로 판별되는 이상 징후는 자동으로 여기 쌓여요. 공유할 때 캡처해서 보내주면 원인 파악이 빨라요.

📚 참고 DB · 실제 업로드물 157건 (작품·캐릭터로 검색)

지금까지 실제로 올라간 굿스마일 게시물이야. 작품명·캐릭터·키워드로 검색해서 "이 캐릭터는 예전에 어떻게 썼더라?"를 바로 찾아볼 수 있어. 상품 상세의 "캐릭터 메모"에서도 같은 DB를 바로 검색해서 가져다 쓸 수 있어요.
