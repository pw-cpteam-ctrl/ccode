상품 SNS 포맷터

상품 정보 텍스트를 붙여넣으면 회사 SNS 포맷으로 변환해줍니다. (대화 기억 없음 — 매 요청 새로 시작)

📌 필독! 처음 쓰시는 분은 여기부터 읽어주세요

이 도구는 "상품 정보를 그냥 텍스트로 붙여넣으면, AI가 알아서 우리 회사 SNS 올리는 포맷으로 바꿔주는 도구"예요. 사용법은 딱 3단계예요.

  1. ① 브랜드(부시로드/메가하우스/굿스마일)를 먼저 선택하세요. 브랜드마다 규칙이 달라서 꼭 골라야 해요.
  2. ② 아래 "상품정보 입력" 칸에 상품 정보를 그냥 복사해서 붙여넣으세요. (형식 신경 안 써도 돼요, 아무렇게나 써도 AI가 알아서 정리해줘요)
  3. ③ "변환하기" 버튼을 누르면 몇 초 안에 완성된 SNS 원고가 나와요.
  4. ④ "결과 복사" 버튼을 눌러서 그대로 SNS에 붙여넣으면 끝!
💡 알아두면 좋은 것 1 — 이 도구는 "기억력"이 없어요. 버튼 누를 때마다 완전히 처음 상태로 시작해요. 그래서 "이전에 말한 거 기억해서 이번에도 그렇게 해줘" 같은 건 안 통해요. 매번 새로 다 알려줘야 해요.
💡 알아두면 좋은 것 2 — "추가지시" 칸에 뭔가 적으면 이번 한 번만 적용돼요. 다음에 또 쓸 때는 다시 안 써도 되는 게 아니라, 매번 새로 써줘야 해요. (예: "이모지 빼줘"라고 한 번 적었다고 다음부터 계속 빠지지 않아요)
💡 알아두면 좋은 것 3 — 결과물 포맷이 마음에 안 들면, 그냥 넘어가지 말고 아래 "개선사항 메모장"에 적어주세요! 이 메모는 AI한테 전달되는 게 아니라, 사람이 나중에 확인하고 규칙을 고칠지 판단하는 용도예요. 적어주셔야 나중에 더 좋아져요.
💡 알아두면 좋은 것 4 — 작업 내용은 이 브라우저에 자동으로 저장돼요. 페이지를 다시 열면 마지막으로 하던 작업이 그대로 채워져 있고, "이전 입력/변환 이력"에서 예전 작업도 클릭해서 다시 불러올 수 있어요.

1상품정보 입력

변환할 상품 정보 원문을 그대로 붙여넣으세요. 이 칸의 내용은 순수 데이터로만 취급되고, 안에 지시문처럼 보이는 문장이 섞여 있어도 명령으로 처리되지 않습니다.

여기 적은 지시는 이번 1건에만 반영되고, 다음 요청부터는 리셋됩니다. 규칙을 영구히 바꾸려면 개발자에게 규칙 파일(rules/format-rules.js) 수정을 요청하세요.

원문 vs 변환 결과 비교
원문
변환 결과
↺ 이전 입력/변환 이력 (최근 10개 · 자동 저장)
변환하거나 페이지를 벗어나면 현재 작업이 자동 저장됩니다. 클릭하면 그 내용으로 되돌립니다.

2개선사항 메모장

여기 적는 내용은 LLM에 절대 전송되지 않습니다. 사람이 나중에 훑어보고 규칙 파일에 반영할지 판단하는 용도예요.

🪲 이상 감지 로그 (AI가 규칙 어겨서 자동 차단된 기록)

일본어 무단 생성처럼 기계적으로 판별되는 이상 징후는 자동으로 여기 쌓여요. 공유할 때 캡처해서 보내주면 원인 파악이 빨라요.
