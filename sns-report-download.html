📊 SNS 성과 리포트 대시보드

자사·경쟁사 SNS 실적을 자동으로 모아서 비교 리포트를 만들어주는 도구예요. 설치 파일 하나 받아서 더블클릭 몇 번이면 준비 끝나요.

🔒 로그인 정보는 본인 컴퓨터 안에만 저장되고 어디로도 전송되지 않아요. 이 도구가 수집하는 건 공개된 게시물의 성과 지표(좋아요·조회수 등)일 뿐, 개인정보는 수집하지 않습니다.
 
⬇ 다운로드윈도우용 · 약 200~300MB
이미 설치했다면
대시보드 열기

처음 설치하는 법

  1. 위 다운로드 버튼 클릭 → 설치 파일(.exe) 바로 받아짐
  2. 받은 파일 더블클릭 → 설치 마법사에서 다음 · 다음 · 완료
  3. 바탕화면에 생긴 "SNS 리포트 실행" 아이콘 더블클릭
  4. 검은 창이 뜨고 잠시 후 브라우저가 자동으로 열리면서 대시보드 화면이 나와요
  5. 트위터/인스타그램 각각 "로그인하기" 버튼 클릭 → 뜬 브라우저 창에서 직접 로그인 → "로그인 완료했어요" 클릭 (처음 한 번만)
⚠️ 검은 창은 도구가 켜져 있는 동안 계속 열어두세요 — 창을 닫으면 도구도 같이 꺼져요.
💡 처음 실행할 때 윈도우 방화벽 허용 팝업이 뜰 수 있어요 — [허용]을 눌러주세요 (정상이에요).

나중에 업데이트할 때

도구가 개선되면, 바탕화면의 "SNS 리포트 업데이트" 아이콘을 더블클릭하기만 하면 돼요. 바뀐 부분만 몇 초 만에 자동으로 받아옵니다 — 다시 설치할 필요 없어요.

잘 안 될 때

증상: 검은 창은 떴는데 브라우저가 저절로 안 열림

→ 주소창에 직접 http://localhost:4848을 입력하세요.

증상: 검은 창에 빨간 글씨가 뜨거나, 화면이 멈춘 채 움직이지 않음

→ 그 화면을 캡처해서 관리자에게 보내주세요.
