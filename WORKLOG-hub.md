### 카드 이름 글자 크기 확대 요청
- **무슨 문제/요청이 있었나**: "인스타 카드생성기", "이벤트추첨기" 카드 이름 글자(14px)가 작으니 키워달라는 요청. 단 줄바꿈은 생기지 않았으면 좋겠고, 줄바꿈이 생기면 가로 여백을 늘리는 식으로 대응해달라는 조건
- **확인해보니**: 8개 카드 이름 전부 같은 CSS 클래스(`.name`)를 공유해서 하나만 키워도 전체 카드에 적용됨. 실제로 Playwright로 브라우저를 띄워 렌더링을 확인해보니, 16px까지는 데스크톱(900px)·모바일 1열 전환 폭(420px) 양쪽에서 8개 카드 이름이 전부 1줄을 유지했고, 18px부터는 긴 이름("SNS 성과 리포트 대시보드" 등)이 2줄로 넘어감. 375px·320px처럼 아주 좁은 폭에서는 원래 14px에서도 일부 긴 이름이 2줄이었던 것을 확인 — 즉 16px로 올려도 새로 생기는 줄바꿈은 없음
- **어떻게 처리했나**: `.name`의 `font-size`를 14px → 16px로만 변경 (18px은 줄바꿈이 발생해서 채택하지 않음). 가로 여백을 늘리는 방식은, 16px에서 이미 목표 두 카드가 문제없이 1줄이라 별도로 손대지 않음
- **결과**: Playwright로 900px/420px/375px/320px 네 폭에서 전수 확인 후 `main`에 커밋·push 완료(`6659f6c..1bbcea0`)
