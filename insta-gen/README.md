# 인스타 카드 생성기

사진 **1장**을 올리면 인스타용 카드(1080×1350)를 만들어 주는 단일 웹 도구입니다.
`index.html` 하나로 동작 — 빌드/서버 없이 브라우저로 바로 엽니다.

## 레이어 구조 (3층)

```
1층  텍스트     ← 자동 추출(OCR) 후 editable 수정
2층  사진       ← 업로드한 사진, 웹에서 크롭(드래그/줌)만 가능
3층  배경지     ← 같은 원본 이미지를 불투명 70% + 흐림으로 깔기
```

로고·한자·배지·누끼 **없음**. 폰트는 **Pretendard (300~800)** 로 통일.

## 사용법

1. `index.html` 을 브라우저로 연다 (더블클릭).
2. 사진 1장 업로드 → 배경지(3층)와 사진(2층)에 자동 적용.
3. **텍스트가 OCR로 1차 자동 추출**되어 필드에 채워짐 → 틀린 곳만 수정.
4. 사진을 **드래그해서 위치 조정**, 휠/슬라이더로 **확대**(크롭).
5. 배경 불투명·흐림 슬라이더로 미세 조정.
6. **PNG 다운로드** 클릭 → `insta-card.png` 저장.

## 토큰 / 속도

- **OCR은 브라우저에서 실행(Tesseract.js)** → API를 안 거치므로 **토큰 0**.
- 자동 추출이 부정확하면 editable 필드에서 2~3초 수정. 단일 이미지 기준 처리 1분 미만.
- 오프라인이면 OCR은 건너뛰고 텍스트를 직접 입력(나머지 기능은 그대로 동작).

## 외부 의존 (CDN)

| 용도 | 출처 |
|---|---|
| Pretendard 폰트 | `cdn.jsdelivr.net/gh/orioncactus/pretendard` |
| 브라우저 OCR | `cdn.jsdelivr.net/npm/tesseract.js` |

네트워크가 막히면 폰트는 시스템 폰트로, OCR은 수동 입력으로 폴백됩니다.
완전 오프라인이 필요하면 두 파일을 로컬에 받아 경로만 바꾸면 됩니다.

## 커스터마이즈 포인트 (index.html 내부)

- `FRAME` — 사진 프레임 위치/크기/모서리
- `render()` 의 `fillText` 좌표·`setFont(weight,size)` — 텍스트 배치/굵기
- 상/하 스크림 그라데이션 — 텍스트 가독성
